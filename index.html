Impact Tracker

A web application to track your environmental impact

A Java web application which allows you to enter positive impact commitments and keep track of the total estimated impact you are having on the world.

Users can create an account and app commitments to certain actions over a time period. The impact calculator will then give an estimate of the total impact.

Technologies used: Spring, JDBC, Thymeleaf, Spring Security, JWT, Bootstrap, MySQL and a RESTful API to allow access for a mobile applicaton.

  • Date: December 2018
  • Languages: Java, HTML, CSS, Javascript

Impact Tracker App

Android application to access the Impact Tracker web application

An Android application written using Java to connect to the Impact Tracker web application RESTful APU.

Users can login to view their impact statistics as well as add, edit and delete commitments.

  • Date: December 2018
  • Language: Java

Tic-tac-Toe App

A TicTacToe Android application

A simple Tic Tac Toe Android application which allows users to play a local two player mode and a single player mode against a computer. There are three different difficulties to choose from for single player: easy, medium and impossible.

Utilises the Minimax algorithm for the AI.

  • Date: October 2018
  • Language: Java

Mastermind App

A Mastermind Android Application

An Android version of the classic game Mastermind. Players try to decipher a 4 colour code by making guesses.

After each guess the user is told how many colours are correct and in the correct place, and how many are correct but in the wrong place.

  • Date: April 2020
  • Language: Java
